elk加密，感谢梦龙小兄弟提供的文档哈哈哈

Elk加密

cd bin/ && ./elasticsearch-certutil ca 敲回车，不需要输入密码，否则启动会报错

./elasticsearch-certutil cert –ca elastic-stack-ca.p12 敲回车，不需要输入密码，否则启动会报错

cd ../

cp elastic-* config/

cd config/

ls

chown elasticsearch elastic-certificates.p12 elastic-stack-ca.p12

ll

vi elasticsearch.yml

xpack.security.enabled: true

xpack.security.transport.ssl.enabled: true

xpack.security.transport.ssl.verification_mode: certificate

xpack.security.transport.ssl.keystore.path: /usr/local/elasticsearch/elasticsearch-7.6.2/config/elastic-certificates.p12

xpack.security.transport.ssl.truststore.path: /usr/local/elasticsearch/elasticsearch-7.6.2/config/elastic-certificates.p12

（路径必须绝对路径）

su elasticsearch

启动es

ps -ef | grep elasticsearch

./elasticsearch-setup-passwords interactive 设置用户名密码

（./elasticsearch-setup-passwords auto 自己生成密码）
