linux一些基础

  view

基础

命令:pwd 查看当前目录

格式：pwd
选项：-p 查看当前目录的真实路径
命令:cd 切换工作目录
格式：cd +目录名
例” cd /home 以根号开头的叫绝对路径 反之则是相对路径
看情况 在哪个路径方便的时候使用不同的路径
命令字+选项+参数
每个目录下都会有两个特殊目录 “.”和·”..“
.代表本身
..代表父级

[root@localhost~]#

root 当前用户名
localhost 当前主机名
~ 当前所在目录
＃ 当前用户标识符
root 超级（管理员)用户
~ 当前用户家目录
＃ 超级用户标识符
$ 普通用户标识符

命令: ls 查看目录下的文件
格式： ls + 目录名
选项：
-l 以长格式详细查看
drwxr-xr-x. 2root root 26 8月 17 22：15 date1
d目录类型
rwxr-xr-x权限 user group other
rwx r-x r-x
r可读权限 w可写权限 x可执行权限 -占位符

0— 1–x 2-w- 3-wx 4r– 5r-x 6rw- 7rwx

.是否受到内核保护的文件
2该目录下的以及子目录的个数
root root 属主 属组
26 文件占磁盘空间的大小
8月17 22：15 最近一次修改文件的时间
data1 文件名
-h 以单位的形式显示 b kb mb gb tb
-a 查看所有文件包括隐藏文件
隐藏文件 在Linux 以.开头的文件
-A 查看所有文件，包括隐藏文件除.和..文件以外
-d 只查看其本身不看其子目录及文件
-R 递归查看目录
–color 指定是否使用系统库中的颜色规则（yes/no）
通配符

/ %代表匹配任意字符
？/_代表匹配单个任意字符（不能为空，也可以叫占位符）
命令:alias 别名
格式alias xxx=”命令字+选项“
命令:du 统计文件的空间占比（统计文件大小）
格式：du+文件名
选项：
-a 查看所有子文件的空间占比
-s 只查看自己，不查看其他子文件
-h 以单位形式查看
一般使用du -sh查看文件大小
命令:mkdir 创建目录
格式： mkdir +目录名
选项：
-p递归创建目录 可以在其没有父目录的情况下，将父目录也一次性创建
{1..10}一次创建多个目录
命令:touch 创建文件
格式：touch +文件名
选项
{a..z}一次创建多个文件
命令:cp 复制文件或目录
格式：cp +源文件+目标位置
命令:scp 远程复制
选项
-r 递归复制目录
命令:mv 移动文件或目录
格式：mv +源文件+目标路径
mv mongo.conf /root
命令:ln 链接
格式：ln+源文件+目标路径
ln mongo.conf /root
选项
-s 软连接（源文件必须是绝对路径）
ln -s /usr/local/mongo.cpnf /root
命令:which 查看命令所在位置
格式 ： which +命令字
alias ls=’ls –color=auto’
/usr/bin/ls
/usr/bin/which: no lsd in (/usr/local/bin:/usr/local/sbin:/usr/bin:/usr/sbin:/bin:/sbin:/root/bin)
该字符串不在bin和sbin里面
bin 所有用户都可以使用的命令
sbin只有管理员才能使用的命令
命令:find 根据查找条件查找文件名
格式：find +查找范围+选项+查询条件
例子：find / -name ‘*.cnf’
选项：-name根据文件名称查询
-size根据文件大小查询
-type根据文件类型查询
-user根据用户查找（属主）
-a 同时满足多个条件
-o 满足任一条件
命令：dd 读取文件
dd if=/dev/zero of /wsl/num.cnf bs=10M count=6
if=/dev/zero 从哪个文件获取 /dev/zero 里边是无限的二进制
of=/wsl/num.cnf 复制到哪个文件去
bs=10M 每次复制10M数据
count=6 复制6次
d 目录
f 普通文件
l 符号链接
例子
find /wsl -user root
find ./ -size 60M
find / -name ‘.cnf’
find /wsl -type d
vim 文本编辑器
命令行模式 ESC 进入
输入模式 a光标后一位 i当前光标除插入 o换行插入
末行模式 ：冒号进入
w 保存但不退出
q 不保存退出
wq 保存并退出
wq！ 强制保存并退出

vim 文件名
i 输入模式 往里插入足够的数据
esc 退出当前模式进入命令模式
命令:cat 查看文件内容
格式：cat+文件名
命令字：echo向屏幕上输出一段字符串
格式：echo+字符串
命令字：>输出重定向 将输出到屏幕的内容 保存到指定文件中（覆盖原文件）
格式： >+文件名
命令字：>>追加重定向 将输出到屏幕的内容 追加保存到指定文件末行
格式：>>+文件名
命令:head 查看文件开头部分的内容（默认10行）
格式：head+-n +文件名
例：head -4 test
head -n 4 test
命令:tail 查看文件末尾部分内容（默认10行）
格式：tail +-n +文件名
例： tail -4 test
tail -n 4 test
命令:| 管道符 将上一条命令得到得结果 给下一行做参数
例：head -8 test | tail -6 查看文件的第三到八行
命令:wc 统计文件参数
格式： wc +文件名
选项： -l 查看文件行数 以空格键 每行的开头默认有一个空格键
-w 查看单词个数 字符串之间以空格间隔或回车间隔记作一个单词
-c 查看字节数 字母数字以及标点符号占一个字节，中文占三字节
命令:grep 根据条件过滤出文件内容
格式： grep +选项 条件 目标文件
条件：
1.一定要用引号括起来
2.’^…’ 以开头的文件内容 ‘…$’以结尾的内容
3.‘^’ 空行 ‘^#’以#开头的行注释行
例：grep ‘^2’ 1查看文件1中的以2开头的行
grep ‘a$’ 1查看文件1中以a结尾的行
grep ‘^$’ 1查看文件1中的空行
grep ‘^#’ 1查看文件1中以#开头的行
选项：
-i 不区分大小写
-v 反转查询 过滤掉不需要的内容
命令:gzip/bzip2 压缩文件
格式： gzip/bzip2 +选项 +文件名
选项：
-9 最高级别的压缩方式
-d解压缩
gzip - 9 1 把文件1以gzip的方式进行压缩
gzip -d 1 将文件1以gzip的方式进行解压
命令:tar 归档压缩
选项：
z 以gzip 的方式进行压缩或解压
j 以bzip2的方式进行压缩或解压
c 归档 打包
x从存档展开文件
v进程可视化
f必须要有，只能放在所有选项最后
p展开所有保护信息
t列出存档中文件的目录
C指定解压路径
归档压缩
tar czf 1.tar.gz 1 2 将文件1和2使用gzip的方式压缩，并归档在 1.tar.gz的包中
tar cxf 1.tar.gz -C we将tar包1.tar.gz 解包至指定目录we中
命令:rpm
常用选项：
-qa 查询所有已安装的软件包
-qi 展示包信息,包括名字,版本,以及描述
-ql 列出该包的文件
-qf 查询拥有文件的包
-qc 只列出配置文件
-qd 只列出文档文件
-i 安装指定文件包
-h 显示安装进度
-v 显示更新的详细信息
rpm ivh 包名
-e 卸载指定的软件包
-U 将指定已经安装过的软件包更新至最新版本，如果未安装则不更新
-F 将指定已经安装过的软件包更新至最新版本，如果未安装，直接安装最新版本
-qpi 查询未安装的软件包的详细信息
-qpl 列出指定的未安装软件包包含的所有文件
-qpc 列出指定的未安装软件包的所有配置文件
例子
rpm -ivh zip -3.0-11.el7.x86_64.rpm
安装zip软件包，并在安装的过程中显示安装进度和详细信息
rpm -qa 查看所有已经安装的软件包
rpm -qa|grep zip 查看当前系统是否已经安装zip
rpm -e dhcp 卸载软件 dhcp
rpm -qi dhcp 查看安装过的软件包的详细信息
rpm -qc dhcp 查看安装过的软件包的所有配置信息
rpm -ivh zip -3.0-11.el7.x86_64.rpm –nodeps 无视依赖包安装
rpm -e dhcp –nodeps 无视依赖关系卸载
yum库
1.必须在/etc/yum.repos.d下的.repo结尾的文件才会生效
2.本地yum只允许yum.repos.d下存在唯一一个以.repo结尾的文件
[root@localhost yum]# cd /etc/yum.repos.d/
[root@localhost yum.repos.d]# mkdir old
[root@localhost yum.repos.d]# mv CentOS-* old
[root@localhost yum.repos.d]# vim yum.repo
[yum] yum名称 文件名yum.repo .前面的字符串一致
name=12111 描述随便写
baseurl=file：///test 本地yum库所在路径 file：//本地yum固定
enabled=1 是否打开yum源
gpgcheck=0 是否进行密钥验证
mkdir /test 创建挂载目录
mount /dev/cdrom /test 将光盘挂载到/test上
yum list 查看yum列表
yum clean all 清空缓存
yum -y install gcc* httpd 安装httpd
yum -y remove httpd 卸载httpd
源码安装
1.对安装包进行解压
[root@localhost ~]# tar xzf cmake-2.8.7.tar.gz -C /usr/src/
cd /usr/src/cmake-2.8.7/
ls 查看该目录下是否有Makefile文件
该文件通过make工具编译时需要使用文件
2生成Makefile文件
./configure 该步骤用于对程序进行安装目录选择，功能选择，
然后编译所需要的Makefile文件
3编译
make#编译是指将程序的源代码文件编辑成计算机可识别的二进制文件
，编译成功后，该程序才算是一个完整的安装程序
4安装
make install 将第三步可安装程序安装到系统中
