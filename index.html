关于PS

命令字：ps 静态查询进程信息
常用选项
-a查看所有进程
-u以用户为主查看进程信息
-x查看进程详细信息
-e查看所有运行的进程
-f以长格式的形式列出进程信息
ps -aux 查看当前终端运行的所有进程的详细信息
ps -ef 查看所有进程的详细信息
USER 表示启动这个进程的用户
PID 进程ID，每个进程的标识符
PPID 该进程的父进程ID
%CPU 该进程对CPU的使用率
%MEM 该进程对内存的使用率
VSZ 该进程使用虚拟内存容量
RSS 该进程使用物理内存容量
TTY 显示进程运行的终端，虚拟终端为pst 终端标识符为tty 如果程序不是在终端下运行显示？
STAT 当前程序的运行状态 R表示运行 S表示休眠 s表示该进程式父进程
START 表示该进程的运行时间
TIME 表示该进程使用CPU的总时间
COMMAND 表示该进程通过哪个命令启动的
命令字：top 动态持续查看进程信息，每5秒刷新一次进程信息
进程运行状态切换指令
&将进程放入后台运行
ctrl +z：将进程挂起，放入后台暂停运行
ctrl +c: 终止正在运行的进程
jobs：查看后台的进程列表，每个任务进程都有一个任务编号，编号从1开始
fg：将指定的编号进程从后台调入前端运行
bg：将指定编号暂停运行的后台进程，在后台继续运行
杀死进程的命令
kill杀死进程的id号 强制杀死使用选项-9
killall杀死指定进程的进程名 强制杀死 -9
pkill 可以通过进程ID号和进程名杀死一个进程
